Welcome to Xenqtt

Xenqtt is a support library and application suite that offers clients powerful and innovative features for working in an MQTT-enabled ecosystem. These features help to address certain gaps and needs that have come up within this relatively new and increasing popular communication protocol. Specifically, Xenqtt provides the following:

  • An MQTT client that operates in both blocking and non-blocking modes
  • An HTTP to MQTT gateway facilitates communication in an MQTT-enabled ecosystem via HTTP (PENDING)
  • A clustering proxy that allows a group of clients to act as a single MQTT client, transparently (PENDING)
  • A mock MQTT broker that is useful for testing and debugging purposes

This website provides a detailed overview of Xenqtt, information on how to download and use the library in your projects, example code, frequently asked questions, and other related resources. We encourage you to browse around!
